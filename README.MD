# Swift Translator Automation Testing - IT23242104

This project automates 39 test cases for the Swift Translator website using Playwright.

## How to Install
1. Clone the repository: `git clone https://github.com/Vihanga09/IT23242104-Swift-Translator-Automation.git`
2. Install dependencies: `npm install`
3. Install Playwright browsers: `npx playwright install`

## How to Run Tests
To run all tests in headed mode with a single worker (recommended):
`npx playwright test --project=chromium --headed --workers=1`

## Test Report
To view the test results:
`npx playwright show-report`
